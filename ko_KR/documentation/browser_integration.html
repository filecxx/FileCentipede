설치 방법 브라우저 확장


이 웹 브라우저 확장을 사용하여 브라우저의 기본 다운로드 도구를 대체할 수 있습니다.

확장 기능은 웹 페이지에서 비디오와 오디오를 캡처할 수 있으며, m3u8 HTTP 스트림 비디오 다운로드 및 자동 병합을 지원하며, 대부분의 암호화 비디오 자동 암호 해독을 지원하며 웹 페이지에서 모든 마그넷 링크를 가져올 수 있습니다.






설치 방법 Firefox 브라우저 확장

열기:

about:addons



떨구기 firefox.xpi 파일 에서



설치 방법 Chrome 브라우저 확장

열기:

chrome://extensions/





사용함 개발자 모드



떨구기 chrome.crx 파일 에서

또는:

Unzip chrome.zip
