주소 만들기

Javascript

웹 사이트를 실행하는 경우입니다.

웹 페이지에 이 스크립트를 삽입하거나 웹 페이지에 스크립트 코드를 삽입합니다. 그러나 이 웹 사이트가 온라인 상태가 아닌 경우 웹 사이트에 스크립트를 넣는 것이 더 좋은 방법입니다.

<script1 src="//filecxx.com/script/create_filec_address.js"></script1>
스크립트 다운로드

사용법:

JSON을 create_filec_address 함수에 전달합니다. json 필드 참조 JSON 

create_filec_address({uri:"http://...."});

Sample:

fileu:0eyJ1cmkiOiJodHRwczovL2dpdGh1Yi5jb20vZmlsZWN4eC9GaWxlQ2VudGlwZWRlL3Jhdy9tYWluL3JlbGVhc2UvY2hyb21lLmNyeCJ9

사용자로부터 실제 주소를 숨기거나 사용자 다운로드에 대한 제한 사항을 제공하려면 소프트웨어에서 \"주소 만들기\" 도구를 사용해야 합니다. 그러면 암호화된 주소를 생성할 수 있습니다.

(Javascript에서 암호화된 주소를 만들 수 없습니다.)

JSON: 작업 추가

JSON을 주소로 사용하여 작업을 추가할 수 있지만 JSON 문자열은 한 줄에 있어야 합니다.

키 유형 설명
uri 문자열 주소

HTTP / HTTPS / Stream 선택 사항

키 유형 설명
method 문자열 GET, POST ... You know the thing
catalog 문자열 카탈로그
page_url 문자열 페이지 url
user_agent 문자열 User Agent
cookies 문자열 Cookies
headers 문자열 HTTP Headers
content_type 문자열 Content Type (for HTTP POST)
data 문자열 HTTP request data
connect_timeout 번호 연결 시간 초과
max_connections 번호 최대 연결
max_redirect 번호 최대 리디렉션
max_retry 번호 최대 재시도
file_name 문자열 파일 이름
speed_limit_download 번호 다운로드 속도 제한
auto_rename_file bool 파일 이름 자동 변경
ignore_remote_modified 번호 수정된 원격 무시
ignore_server_busy 번호 서버 사용 중 무시
ignore_http_error 번호 http 오류 무시

FTP / FTPS 선택 사항

키 유형 설명
catalog 문자열 카탈로그
page_url 문자열 페이지 url
connect_timeout 번호 연결 시간 초과
max_connections 번호 최대 연결
max_retry 번호 최대 재시도
file_name 문자열 파일 이름
user 문자열 사용자 이름
pass 문자열 암호
speed_limit_download 번호 다운로드 속도 제한
speed_limit_upload 번호 업로드 속도 제한
auto_rename_file bool 파일 이름 자동 변경
ignore_server_busy 번호 서버 사용 중 무시

Magnet / HASH 선택 사항

키 유형 설명
catalog 문자열 카탈로그

max_connections 번호 최대 연결
