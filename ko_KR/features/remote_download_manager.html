원격 다운로드 관리자

File centipede는 원격 파일 다운로드를 위한 기본 사용자 인터페이스를 제공했습니다.

그것은 원격이라는 기본 메뉴에 있습니다.

사용법이 매우 간단합니다. 원격 호스트의 IP/도메인 주소를 입력한 다음 방금 추가한 메뉴 항목에서 호스트를 클릭하면 됩니다.

컴퓨터를 연결할 수 있도록 하려면 설정 -> 서비스를 열고 원격 접근을 사용 가능으로 전환하고 암호를 설정해야 합니다.

명령 행

filec remote_access on

filec service_port 10111

filec service_password your_password
