WebDAV 파일 관리자

File Centipede는 WebDAV 파일 관리자이기도 합니다. 디렉터리 또는 전체 WebDAV 사이트를 포함하여 WebDAV 서버에서 파일을 다운로드하고 업로드할 수 있습니다.

이 기능은 매우 사용하기 쉽고, 많은 설정을 입력할 필요가 없으며, 주소 표시줄에 호스트 주소를 입력하기만 하면 됩니다.

디렉터리 파일 나열 속도를 최대화하려면 다음과 같이 하십시오. File Centipede는 방문한 모든 디렉터리 파일을 캐시할 수 있으며, 다음에 나열할 때 동일한 디렉터리가 로컬 디렉터리를 여는 것과 같습니다.
