비디오 다운로드기

File Centipede는 99% 이상의 웹 사이트에서 동영상을 다운로드할 수 있습니다.

또한 Javascript에서 비디오 주소를 분석하고 Javascript 폐쇄에서 해독된 비디오 키를 추출할 수 있습니다. 일부 특수 웹 사이트의 경우 파일 지네만 비디오를 다운로드할 수 있습니다.

그것은 웹 페이지에서 비디오나 오디오를 다운로드하는 두 가지 쉬운 방법을 제공했습니다.

1: 동영상 옆에 있는 플로팅 표시줄에서 다운로드합니다.

2: 팝업 창에서 다운로드합니다.
