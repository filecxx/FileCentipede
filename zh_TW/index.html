描述

File centipede 是一款一體化網際網路檔案上傳/下載管理器、BitTorrent 用戶端、WebDAV 用戶端、FTP 用戶端和 SSH 用戶端。它快速, 可客製化, 使用者友善, 支援多種通訊協定, 同時也包含了很多有用的輔助工具如: HTTP請求器, 檔案合併工具, 編碼/解碼工具等. 和網頁瀏覽器進行整合, 你可以從網頁上下載音訊和視訊, 甚至是加密視訊。
