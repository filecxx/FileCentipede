Wie installiert man Browser-Erweiterung


Diese Webbrowser-Erweiterung kann verwendet werden, um das Standard-Download-Tool des Browsers zu ersetzen.

Die Erweiterung kann Videos und Audio von der Webseite erfassen, sie unterstützt m3u8 HTTP-Stream-Video-Download und Auto-Merge, sie unterstützt das automatische Entschlüsseln der meisten verschlüsselten Videos, und sie unterstützt das Abrufen aller Magnet-Links von der Webseite.






Wie installiert man Firefox Browser-Erweiterung

Öffnen:

about:addons



reinziehen (hier fallen lassen) firefox.xpi Datei on



Wie installiert man Chrome Browser-Erweiterung

Öffnen:

chrome://extensions/





Aktivieren Entwicklermodus



reinziehen (hier fallen lassen) chrome.crx Datei on

oder:

Entpacken chrome.zip
