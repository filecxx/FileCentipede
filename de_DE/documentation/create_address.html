Adresse erstellen

Javascript

Wenn Sie eine Website betreiben.

Fügen Sie dieses Skript in Ihre Webseite ein, oder setzen Sie den Skriptcode in Ihre Webseite, aber der bessere Weg ist, das Skript auf Ihrer Webseite zu platzieren, falls diese Webseite nicht online ist.

<script1 src="//filecxx.com/script/create_filec_address.js"></script1>
Skript herunterladen

Verwendung:

Geben Sie ein JSON in die Funktion create_filec_address ein, json-Felder siehe JSON 

create_filec_address({uri:"http://...."});

Sample:

fileu:0eyJ1cmkiOiJodHRwczovL2dpdGh1Yi5jb20vZmlsZWN4eC9GaWxlQ2VudGlwZWRlL3Jhdy9tYWluL3JlbGVhc2UvY2hyb21lLmNyeCJ9

Wenn Sie Ihre echte Adresse vor Benutzern verbergen oder einige Einschränkungen für den Benutzerdownload geben möchten, sollten Sie das \"Adresse erstellen\"-Tool aus der Software verwenden, es kann eine verschlüsselte Adresse generieren.

(Javascript kann keine verschlüsselte Adresse erstellen.)
