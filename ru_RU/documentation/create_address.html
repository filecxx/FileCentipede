Создать адрес

Javascript

Если вы имеете свой веб-сайт

Вставьте этот скрипт на свою веб-страницу или поместите код скрипта на свою веб-страницу, лучший способ - разместить скрипт на своем веб-сайте, если этот веб-сайт не подключен к сети.

<script1 src="//filecxx.com/script/create_filec_address.js"></script1>
Скачать скрипт

Использование:

Пропустить JSON в функцию create_filec_address, увидеть поля JSON JSON 

create_filec_address({uri:"http://...."});

Sample:

fileu:0eyJ1cmkiOiJodHRwczovL2dpdGh1Yi5jb20vZmlsZWN4eC9GaWxlQ2VudGlwZWRlL3Jhdy9tYWluL3JlbGVhc2UvY2hyb21lLmNyeCJ9

Если вы хотите скрыть свой реальный адрес от пользователей или ввести некоторые ограничения для загрузки пользователями, вам следует использовать инструмент \\\\\\\\\\\\\\\"создать адрес\\\\\\\\\\\\\\\" в программе, он может сгенерировать зашифрованный адрес.

(Javascript не может создать зашифрованный адрес.)

JSON: Добавить задачу

Вы можете использовать JSON как адрес для добавления задачи, но помните, строка JSON должна быть в одну линию.

Клавиша Тип Описание
uri Цепочка Адрес
