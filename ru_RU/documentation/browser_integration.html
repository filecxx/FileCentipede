Как установить Расширение браузера


Это расширение для браузера можно использовать для замены стандартного инструмента загрузки в браузере.

Расширение может захватывать видео и аудио с веб-страницы, поддерживает загрузку и автоматическое объединение видео потока m3u8 HTTP, поддерживает автоматическую расшифровку большинства зашифрованных видео, поддерживает получение всех magnet ссылок с сайтов.






Как установить Firefox Расширение браузера

Открыть:

about:addons
