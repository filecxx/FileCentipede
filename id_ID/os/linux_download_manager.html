Manajer unduh Linux

File Centipede adalah manajer unduh Linux terbaik.

Ini adalah satu-satunya yang mendukung semua protokol utama dan manajer unduh video sniffer untuk Linux.

Ini juga berisi banyak alat bantu seperti penggabungan file, checksum, pengkode base64, pengkode URI, dan magnet-ke-torrent......
Anda juga dapat menggunakannya untuk mengelola file jarak jauh seperti WebDAV, FTP, dan SSH.
Sangat mudah untuk menginstalnya, hanya perlu satu klik, dan tidak memerlukan pustaka dinamis lainnya, bahkan Glibc.
