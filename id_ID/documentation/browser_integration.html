Cara menginstal Ekstensi penjelajah


Ekstensi penjelajah web ini dapat digunakan untuk menggantikan alat unduh default dari penjelajah.

Ekstensi ini dapat menangkap video dan audio dari halaman web, mendukung unduhan video aliran HTTP m3u8 dan penggabungan otomatis, mendukung dekripsi otomatis untuk sebagian besar video yang terenkripsi, dan mendukung pengambilan semua tautan magnet dari halaman web.






Cara menginstal Firefox Ekstensi penjelajah

Buka:

about:addons



Jatuhkan firefox.xpi File Menjadi



Cara menginstal Chrome Ekstensi penjelajah

Buka:

chrome://extensions/





Aktifkan Mode pengembang



Jatuhkan chrome.crx File Menjadi

Atau:

Ekstrak chrome.zip
