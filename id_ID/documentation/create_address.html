Javascript

Jika Anda menjalankan sebuah situs web.

Sisipkan skrip ini ke dalam halaman web Anda, atau masukkan kode skrip ke dalam halaman web Anda, tetapi cara yang lebih baik adalah menempatkan skrip di situs web Anda, jika situs ini tidak online.

<script1 src="//filecxx.com/script/create_filec_address.js"></script1>
Unduh skrip

Penggunaan:

Lepaskan JSON ke dalam fungsi create_filec_address, lihat bidang json JSON 

create_filec_address({uri:"http://...."});

Sample:

fileu:0eyJ1cmkiOiJodHRwczovL2dpdGh1Yi5jb20vZmlsZWN4eC9GaWxlQ2VudGlwZWRlL3Jhdy9tYWluL3JlbGVhc2UvY2hyb21lLmNyeCJ9

Jika Anda ingin menyembunyikan alamat asli Anda dari pengguna atau memberikan beberapa batasan untuk unduhan pengguna, Anda harus menggunakan alat \"buat alamat\" dari perangkat lunak, itu dapat menghasilkan alamat terenkripsi.

(Javascript tidak dapat membuat alamat terenkripsi.)

JSON: Tambahkan tugas

Anda dapat menggunakan JSON sebagai alamat untuk menambahkan tugas, tetapi ingat, string JSON harus dalam satu baris.

Kunci Tipe Deskripsi
uri String Alamat
