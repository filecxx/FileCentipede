Deskripsi

File Centipede adalah manajer unggah/unduh file internet All-In-One, Klien BitTorrent, klien WebDAV, klien FTP, dan klien SSH. Dirancang untuk cepat, dapat disesuaikan, dan ramah pengguna. Mendukung multi-protokol dan berisi banyak alat bantu berguna seperti permintaan HTTP, penggabungan file, dan pengkode. Dengan integrasi penjelajah, Anda dapat mengunduh audio dan video dari situs web, bahkan video terenkripsi.
