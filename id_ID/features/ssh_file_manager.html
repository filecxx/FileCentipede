Manajer file SSH

File Centipede juga merupakan manajer file SSH, dapat mengunduh & mengunggah file dari server SSH, termasuk direktori atau seluruh situs SSH.

Sangat mudah digunakan, Anda tidak perlu memasukkan banyak pengaturan, Anda hanya perlu memasukkan alamat host ke dalam bilah alamat.

Untuk memaksimalkan kecepatan daftar file direktori, File Centipede dapat menyimpan cache semua file direktori yang telah Anda kunjungi, sehingga saat Anda membuka direktori yang sama berikutnya akan terasa seperti membuka direktori lokal.
