Adres Oluştur

Javascript

Bir web sitesi çalıştırıyorsanız.

Bu scripti web sayfanıza yerleştirin veya script kodunu web sayfanıza koyun, ancak en iyi yol, bu web sitesi yayında değilse, scripti web sitenize koymaktır.

<script1 src="//filecxx.com/script/create_filec_address.js"></script1>
Komut dosyasını indir

Kullanım:

Bir json'u create_filec_address işlevine geçirin, json alanları bkz. JSON 

create_filec_address({uri:"http://...."});

Sample:

fileu:0eyJ1cmkiOiJodHRwczovL2dpdGh1Yi5jb20vZmlsZWN4eC9GaWxlQ2VudGlwZWRlL3Jhdy9tYWluL3JlbGVhc2UvY2hyb21lLmNyeCJ9

Gerçek adresinizi kullanıcılardan gizlemek veya kullanıcıların indirmelerine bazı sınırlamalar getirmek istiyorsanız, yazılımdaki \"adres oluşturma\" aracını kullanmalısınız, bu araç şifrelenmiş bir adres üretebilir.

(JavaScript şifreli bir adres oluşturamaz.)

JSON: Görev ekle

Görev eklemek için adres olarak JSON kullanabilirsiniz, ancak JSON dizesinin tek satırda olması gerektiğini unutmayın.
