Tanım

File centipede, hepsi bir arada internet dosya yükleme/indirme yöneticisi, BitTorrent İstemcisi, WebDAV istemcisi, FTP istemcisi ve SSH istemcisidir. Hızlı, özelleştirilebilir ve kullanıcı dostu olacak şekilde tasarlanmıştır. Çoklu protokolleri destekler ve HTTP istekçisi, dosya birleştirme ve kodlayıcılar gibi birçok yararlı yardımcı araç içerir. Tarayıcı entegrasyonuyla, web sitelerinden ses ve video, hatta şifrelenmiş videolar indirebilirsiniz.
Linux indirme yöneticisi Video indirici BitTorrent istemcisi WebDAV dosya yöneticisi FTP dosya yöneticisi SSH dosya yöneticisi Uzaktan indirme yöneticisi
