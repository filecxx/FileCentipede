文件蜈蚣是一个多合一的互联网文件上传/下载管理器，BitTorrent 客户端，WebDAV 客户端，FTP 客户端和 SSH 客户端。

它旨在快速、可定制且用户友好。

它支持多协议并包含许多有用的辅助工具，例如 HTTP 请求器、文件合并和编码器。

通过浏览器集成，您可以从网站下载音频和视频，甚至是加密视频。

（它是免费的，没有广告）
