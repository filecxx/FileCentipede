How to install Browser extension


This web browser extension can be used to replace the browser's default download tool.

The extension can capture videos and audio from the web page, it supports m3u8 HTTP stream video download and auto-merge, it supports auto decrypting most encryption videos, and it supports getting all magnet links from the web page.






How to install Firefox Browser extension

Open:

about:addons



Drop firefox.xpi File Into



How to install Chrome Browser extension

Open:

chrome://extensions/





Enable Developer mode



Drop chrome.crx File Into

Or:

Unzip chrome.zip



Load unpacked
